Principais formatações

Negrito / destaque

O HTML5 trabalha com semãntica, o que implica em não utilizar tags de forma na sua construção. Posso usar o negrito dessa forma aqui, mas a tag "b" é de forma. Para destacar, o melhor termo é utilizar o "strong" como apresentado na própria palavra. Essa última tag é utilizada para destaque.


Itálico

Essa frase está em itálico, mas está sendo utilizada a tag "i" que não é semântica. A semântica é realizada pela tag "ênfase" ou "em" diretamente na linha de código.


ATENÇÃO

Utilize sempre "strong" ou "em" afinal, como HTML5 foca na semântica, não faz sentido utilizar forma!


Texto Marcado

Realizando uma marcação de texto. Para marcar textos, utilizamos a tag "Mark", mas lembrar que essa tag não é automática no VS Code por enquanto. Estou mudando a cor do "Mark" usando "Style"


Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno. O texto grande não deve ser feito utilizando a tag "big", tendo vista que caso eu queira aumentar um texto, basta formatá-lo em CSS. Porém a tag "small" continua possuindo sentido semântico.
